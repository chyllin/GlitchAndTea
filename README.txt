🛠️ Deployment Instructions (Render)

1. Go to https://render.com and log in or sign up.
2. Click "New" -> "Web Service".
3. Connect your GitHub (or use "Deploy from Zip" if available).
4. Set Environment:
   - Start Command: python app.py
   - Environment: Python 3
5. Add environment variables if needed (not for this basic version).
6. Deploy!

✅ Your site will go live on a public URL.
